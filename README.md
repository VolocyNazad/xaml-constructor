C# source generator that generates a constructor from readonly fields in a class or struct for xaml design mode.

## Installation

- Grab the latest package on [NuGet](https://www.nuget.org/packages/XamlConstructor/).

## Basic usage

The following code:

```csharp
namespace Test;

[XamlConstructor]
public partial class MyClass
{
    protected readonly IServiceProvider _serviceProvider;
    private readonly ILogger<MyClass> _logger;
    private readonly IService _service;
}
```

will generate:

```csharp
// <auto-generated>
//     This code was generated by the XamlConstructor source generator.
//
//     Changes to this file may cause incorrect behavior and will be lost if
//     the code is regenerated.
// </auto-generated>

namespace Test;

partial class MyClass
{
    /// <summary>
    /// XAML Design Mode constructor - all readonly fields will be null!
    /// </summary>
    [Obsolete("This constructor should only be used by XAML markup, and only in DesignMode.")]
    public MyClass()
    {
        this._serviceProvider = null!;
        this._logger = null!;
        this._service = null!;
    }
}
```

## Diagnostics

### XCONS01

The `XamlConstructor` attribute is used on a class that is not partial.

### XCONS02

The `XamlConstructor` generation failed.
